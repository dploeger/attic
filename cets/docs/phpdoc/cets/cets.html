C!ETS  Cool! Edit This Site


Sometimes it's hard to keep it simple.
I was searching for a veeery simple Wiki to use in a portal I was making and found-- nothing. No Wiki I've seen was just  "move the script here, set some code and implement it in your pages like this:...". So I finally wrote me one myself.
Here it is: C!ETS. It is basically a wrapper around database-SQL and HTML to make it easy for you to implement a site,  that can be edited quite simple and using a HTML-WYSIWYG-Editor.  The basic functionality is done using Pear_DB and tinymce.  Please have a look at the manual in docs/manual.
